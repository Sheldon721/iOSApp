# CLQZhihuDaily
iOS版知乎日报（Swift 3.1）

开发工具：Xcode 8.3

开发语言：Swift 3.1

使用的第三方库：Alamofire、SDWebImage、SnapKit、ReactiveCocoa等

功能简介：

1、今日热闻：新闻列表、顶部轮播图片新闻；

2、新闻详情：新闻详情查看，详情页下拉加载上一条、上拉加载下一条新闻；

3、侧边栏
